SDK Examples

Chamaileon SDK Examples

Every example is created in Vanilla.js (without any Javascript frameworks) in order to keep them as simple as possible, with minimal boilerplate code.

You can also fork this repo on Github. You can also follow us on Twitter for updates.

There are two main sections in this document. The first is the basic examples section, in which you will find example codes that use the bare minimal configuration of the plugins. These basic examples are a great starting point.

The second section shows you some advanced examples, with more configuration options that can help you create a very, very smooth experience for your users.
