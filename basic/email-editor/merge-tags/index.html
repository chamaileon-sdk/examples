Email Editor: Merge Tags

This example is focusing on providing merge tags to your users. It configures a button, that will show up in the top toolbar on the right-hand-side when you are editing a text element. If you click on that button, a Javascript hook is called, in which you can implement your custom merge tag dialog.

Search for the icon when you are editing a text element.
